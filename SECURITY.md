# Security Policy

This project should not contain secrets or platform session data.

## Do Not Commit

- API keys, tokens, cookies, or `.env` files
- WeChat, Douyin, Xiaohongshu, YouTube, or other platform login sessions
- QR codes or screenshots that expose login state
- Paid TTS credentials
- Downloaded copyrighted source videos
- Personal drafts, private publishing history, or private analytics

## Reporting

If you find a security issue in this open-source template, open a GitHub security advisory or contact the maintainer privately.

If you accidentally committed a secret, rotate it immediately. Removing it from the latest commit is not enough once it has been pushed publicly.
